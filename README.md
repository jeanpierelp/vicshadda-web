# Vicshadda Producciones

Sitio responsive para mostrar servicios, paquetes, galeria y redes sociales.

## Sitio publicado

GitHub Pages abre `index.html`, que dirige automáticamente a la versión ubicada en `preview/`.

## Vista previa local

Abrir `preview/index.html` en el navegador. Esta versión usa Bootstrap desde CDN.

## Proyecto Angular

```powershell
npm install
npm start
```

Luego abrir `http://localhost:4200`.
